software-box

The aim of this project is to automate installation of Java development tools for a developer environment. The project consists of a set of MS-DOS scripts that are used to install below listed software.

Steps for using the package
1. Dump all the installers to a remote windows directory. 2. Make sure the Dump directory is accessible from the system where setup us to be done. 3. Edit config.properties change the directory paths accordingly. 4. Invoke the setup.cmd file form dos prompt.
